# FUTURE_ML_01
# Future Interns: machine learning internship

## Task 1: Sales & Demand Forecasting
This project focuses on predicting future sales demand using historical retail data.

### 🛠️ Tools Used
* **Language:** Python
* **Libraries:** Pandas, Matplotlib, Scikit-Learn (Linear Regression)

### 📈 Project Steps
1. **Data Cleaning:** Handled date formatting and removed negative sales (returns).
2. **Analysis:** Used a 7-day rolling average to identify sales trends.
3. **Forecasting:** Built a Linear Regression model to predict the next 30 days of demand.

### 🚀 Key Insight
The model predicts a steady upward trend in sales, suggesting that the business should increase inventory levels to meet upcoming demand.
